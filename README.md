# Conversor-ER-para-AFND

Trabalho para a disciplina de Teoria da Computação que converte Expressões Regulares (ER) para Autômatos Finitos Não Determinísticos (AFNDs) utilizando o Algoritmo de Thompson.

Biblioteca utilizada para gráficos: https://caleb531.github.io/automata/

## Requisitos

1. Instale o Graphviz:
   - https://graphviz.org/download/

2. Instale o Visual C++ Build Tools:
   - https://visualstudio.microsoft.com/visual-cpp-build-tools/
   - Marque o workload **Desktop development with C++**

## Dependências

```bash
pip install automata-lib
pip install coloraide
pip install pygraphviz
pip install Pillow
```
